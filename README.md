# 働 長時間労働原理主義 — WORK OR DIE

黄金に光り輝くグラスモーフィズムデザインの、長時間労働原理主義パロディサイトです。

**※本サイトは100%ジョークサイトです。長時間労働を推奨するものではありません。ちゃんと休みましょう。**

## 構成

- `index.html` — サイト本体(HTML/CSS/JSすべて1ファイル完結)
- `.github/workflows/deploy-pages.yml` — GitHub Pages 自動デプロイ用ワークフロー

## GitHub Pages で公開する手順

1. このブランチを `main` にマージする
2. リポジトリの **Settings → Pages** を開く
3. **Source** で **GitHub Actions** を選択する
4. `main` へのプッシュで自動的にデプロイされ、`https://<ユーザー名>.github.io/work-or-die/` で公開されます

## ローカルで確認する

```bash
# index.html をブラウザで開くだけでOK
open index.html
```
